Save first, then test — it sends a message to yourself using the saved settings.

For Gmail / Google Workspace: host smtp.gmail.com, port 587, user = your full email, password = a 16-character App Password (a normal password won't work). App Passwords only appear after you turn on 2-Step Verification — full step-by-step in the Email setup guide.

AI

Powers PDF statement parsing, receipt reading, and smart categorization. Get a key at console.anthropic.com - costs are typically pennies per statement. Without a key the app still works (CSV import, keyword rules, manual receipt entry).

Prefer to run AI locally instead of Claude? (optional - Ollama)

Backup folder

A copy of every backup is also written here (in addition to the always-on backups next to your database). Use any folder - a OneDrive/Dropbox/Google Drive folder for cloud safety, or an external drive. Leave blank to auto-detect OneDrive. The folder is created if it doesn't exist; saving writes a test backup so you can confirm it works.

Folder watchers

Drop a bank/card statement (PDF/CSV) or a receipt (photo/PDF) into these folders — e.g. your Downloads folder, or wherever your bank/phone already saves them — and ShopBooks checks for new files about once a minute while it's running (not a background service; it only runs while the app is open) and stages them in Review or Receipts exactly like a manual upload. Nothing posts automatically. Leave a field blank to turn that watcher off. The Trips folder powers automatic mileage: your phone drops a tiny event file there when it connects/disconnects from your car's Bluetooth, and detected drives appear on Mileage for approval — phone setup guide in the automatic-mileage guide.

Sync between two computers

Use the same books on two machines (e.g. a Mac and a PC) by syncing through your cloud Backup folder above. ShopBooks keeps a single version-stamped copy there: it pulls newer changes when you open the app and pushes your changes when you close it. It never overwrites newer work silently - if both computers changed the books, it asks you which to keep. Only run the app on one computer at a time, and let your cloud folder finish syncing before switching.
