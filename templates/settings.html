Sync between two computers

Use the same books on two machines (e.g. a Mac and a PC) by syncing through your cloud Backup folder above. ShopBooks keeps a single version-stamped copy there: it pulls newer changes when you open the app and pushes your changes when you close it. It never overwrites newer work silently - if both computers changed the books, it asks you which to keep. Only run the app on one computer at a time, and let your cloud folder finish syncing before switching.
